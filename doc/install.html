Installing opentheory from Source

Installing opentheory from source requires an installation of either MLton, Poly/ML or Moscow ML, as well as standard system tools including GNU Make and Perl.

  1. Download the opentheory.tar.gz file.
  2. Extract it with the command tar xvzf opentheory.tar.gz
  3. There will be a new directory called opentheory. Change to this directory.
  4. To build using MLton:
    • Build with the command make mlton
    • Test with the command bin/mlton/opentheory
  5. To build using Poly/ML:
    • Build with the command make polyml
    • Test with the command bin/polyml/opentheory
  6. To build using Moscow ML:
    • Build with the command make mosml
    • Test with the command bin/mosml/opentheory
  7. If the test results in a usage message, the installation was successful.

If anything goes wrong, you can use the command make clean to clean out any object files.
