What is opentheory?

opentheory is a tool for processing higher order logic theory packages.

Features of opentheory

  • Coded in Standard ML (SML), and ported to the MLton, Poly/ML and Moscow ML compilers.
  • The tool is used to power the OpenTheory project.

Project Status

The latest version is opentheory 1.0 (release 20101214), released under version 2 of the GNU General Public Licence.
