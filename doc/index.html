OpenTheory

The goal of the OpenTheory project is to allow proofs to be shared between the different theorem prover implementations of higher order logic, including HOL4, ProofPower/HOL and hol-light. As a consequence, it is hoped that a standard library of formalized mathematics can be built up for higher order logic.

Project Status

The OpenTheory project has published a file format standard for articles containing statements and proofs of theorems. Articles are not supposed to be self-contained, and may require types, constants and theorems that must be supplied by the theorem proving system reading the article. There is also a set of example articles which can be used to test tools implementing the article standard.

There is active development on tool support for the project. Higher order logic theorem provers involved in the OpenTheory project have to be instrumented to read and write article files. In addition, stand-alone tools are envisaged to normalize articles to use a minimal set of primitive inference rules, eliminate redundancy, create an article signature of required and provided theorems, and join together articles.
